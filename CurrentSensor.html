Code Overview

This script is designed to measure and display voltage, current, and power metrics using the INA219 sensor. The data is shown both on an LCD and the Serial Monitor.

Functions:

  • setup(): Initializes the I2C connection, LCD display, and INA219 sensor. Begins measuring initial voltage.
  • loop(): Continuously gathers data from the INA219 sensor every 2 seconds and updates the displays.
  • getData(): Measures the current, power, and energy from the INA219 sensor and updates the LCD and Serial Monitor with the new readings.

Classes:

  • LiquidCrystal_I2C: Used for communicating with the LCD. Requires the I2C address of the LCD, number of columns, and number of rows.
  • Adafruit_INA219: Used for interfacing with the INA219 sensor to measure voltage and current.

Variables:

  • lcd: Object for the LCD display.
  • ina219: Object for the INA219 current/voltage sensor.
  • current_mA: Current in milliamps measured by the sensor.
  • power_mW: Power in milliwatts calculated from voltage and current.
  • energy_Wh: Energy in watt-hours calculated over time.
  • time_s: Time in seconds since the program started.
  • voltage_V: Voltage in volts as measured by the sensor.

Constants:

No specific constants defined in this script beyond the initialization parameters for the LCD and INA219 sensor.

External Libraries:

  • Wire.h: Allows for I2C communication.
  • LiquidCrystal_I2C.h: Library for the LCD interface over I2C.
  • Adafruit_INA219.h: Library specific to the INA219 sensor for current and voltage measurement.
