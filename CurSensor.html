DEPENDENCIES

This code measures voltage and current using the Adafruit INA219 library. It prints the current, power, and energy values to the Serial Monitor

Global Variables

  • Adafruit_INA219 ina219: An object of the Adafruit_INA219 class to handle the INA219 chip.
  • float current_mA: A floating-point variable to store the current measurement in milliamperes.
  • float power_mW: A floating-point variable to store the power measurement in milliwatts.
  • float energy_Wh: A floating-point variable to store the energy measurement in watt-hours.
  • long time_s: A long variable to store the time in seconds.

Function Definitions

  • void setup(void)
    • Initializes the Serial communication at a baud rate of 9600.
    • Initializes the INA219 sensor using the begin function.
    • Prints a message to indicate that voltage and current measurement is being done.
  • void loop(void)
    • Calls the getData() function.
    • Delays for 2000 milliseconds (2 seconds) before repeating.
  • void getData()
    • Takes measurements for current, power, and energy.
    • Converts the millisecond timestamp to seconds and stores it in the time_s variable.
    • Measures the current using the getCurrent_mA function of the INA219 object and stores it in the current_mA variable.
    • Calculates power in milliwatts by multiplying current with voltage (not defined in the code).
    • Calculates energy in watt-hours by dividing power by 3600 and multiplying it with time in seconds.
    • Prints the current, power, and energy values to the Serial Monitor.
