Dependencies

This code requires the following dependencies:

  • pandas (imported as pd)
  • numpy (imported as np)
  • matplotlib.pyplot (imported as plt)
  • train_test_split from sklearn.model_selection
  • MinMaxScaler from sklearn.preprocessing
  • Sequential from tensorflow.keras.models
  • LSTM from tensorflow.keras.layers
  • Dense from tensorflow.keras.layers

Load Data

Loads battery health data from a CSV file using the pd.read_csv() function. Specify the path to the CSV file in the file_path variable.

Preprocessing

Define input features (X) and target variable (y) from the loaded data. Normalize the data using Min-Max scaling to ensure features are in the range [0, 1].

Train-Test Split

Split data into training and testing sets using the train_test_split() function. Set the test size to 20% and random state to 42.

Reshape Data

Reshape data to fit the LSTM model's input shape: (samples, time steps, features). Time steps are typically set to 1.

Build LSTM Model

Construct an LSTM model with 50 units followed by a Dense layer with 1 unit. Set the output layer's activation function to 'linear'.

Compile Model

Compile the model using 'adam' as the optimizer and 'mean_squared_error' as the loss function.

Train Model

Train the model with specified epochs and batch size, using the training data and also provide the validation data.

Evaluate Model

Evaluate the model's performance using the testing data and store the loss value.

Make Predictions

Use the model to make predictions on the testing data and store the results.

Inverse Transform

Inverse transform the predictions and actual values to convert them back to the original scale.

Plot Results

The results are plotted using matplotlib.pyplot. Two plots are created:

  1. Scatter Plot: A scatter plot comparing the predicted SOH values (predictions_inv) and the true SOH values (y_test_inv) is created. The data points are marked with different markers to distinguish between the predicted and true values. The x-axis represents the data points, and the y-axis represents the SOH values.
  2. Line Plot: A line plot showing the true SOH values (y_test_inv) and the predicted SOH values (predictions_inv) is created. The x-axis represents the data points, and the y-axis represents the SOH values. The x-axis limits are adjusted for better visualization.
