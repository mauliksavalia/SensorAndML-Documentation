Code Documentation

Functions:

  • setup(): Initializes the sensors and LCD display. Sets the initial message on the LCD.
  • loop(): Continuously checks the sensor readings and updates the LCD and Serial Monitor with the drone detection status.

Classes:

  • LiquidCrystal_I2C: Used for communicating with the LCD. Requires the I2C address of the LCD, number of columns, and number of rows.

Variables:

  • irSensorPin1, irSensorPin2: int pins connected to IR sensors.
  • piezoSensorPin1, piezoSensorPin2: int pins connected to Piezo sensors.
  • piezoThreshold: int value above which piezo is considered "triggered".
  • lcd: LiquidCrystal_I2C object for the LCD display.

Constants:

  • Sensor pins (irSensorPin1, irSensorPin2, piezoSensorPin1, piezoSensorPin2) are assigned to specific board pins and are not meant to change throughout the program.
  • piezoThreshold is set based on sensitivity requirements and may be adjusted for different environments or sensors.

External Libraries:

  • LiquidCrystal_I2C: Handles the interfacing with the LCD display over the I2C communication protocol.
