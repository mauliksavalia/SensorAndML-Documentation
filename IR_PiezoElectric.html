DESCRIPTION

This code is written for an Arduino to measure the distance and detect motion using an infrared (IR) sensor and a piezo sensor. The code sets up the pins in the Arduino and initializes the serial communication. It then enters the loop and reads the status of the IR sensor and the piezo sensor. If both sensors are triggered, it outputs a message indicating that motion is detected. Otherwise, it outputs a message indicating no motion.

VARIABLES

distance - int

Description: The distance threshold for the IR sensor. The IR sensor is triggered if the reading is below this distance with Initial value: 1

irSensorPin - int

  • Description: The pin number for the IR sensor.

piezoSensorPin - int

  • Description: The pin number for the piezo sensor.

piezoThreshold - int

  • Description: The threshold value for the piezo sensor. The piezo sensor is triggered if the reading is above this threshold.

setup()

  • Description: Initializes the Arduino by setting up the pins and starting the serial communication.

loop()

  • Description: The main loop of the code. Reads the status of the sensors and outputs appropriate messages based on the sensor readings.
