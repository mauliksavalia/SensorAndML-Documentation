Overview

This document provides the setup and operational details for a system utilizing two infrared (IR) sensors to detect the presence of objects, specifically drones. It explains how the sensors are defined and configured within the system.

Pin Definitions

  • irSensor1 (int): Pin number connected to the first IR sensor (4).
  • irSensor2 (int): Pin number connected to the second IR sensor (5).

Setup()

  • Configures irSensor1 and irSensor2 as input pins.
  • Initializes the serial communication at a baud rate of 9600.

Loop

  • Reads the status of the IR sensors and stores them in sensor1Status and sensor2Status variables (both of type bool).
  • If both sensor1Status and sensor2Status are true (sensors are triggered), it prints "Drone Detected!" using serial communication.
  • If either sensor1Status or sensor2Status is false (no sensors triggered), it prints "No drone detected." using serial communication.
  • Delays the program execution for 200 milliseconds.
