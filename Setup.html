Introduction

Welcome to the Smart Recharging Platform for Drones and Ultracapacitors project! This innovative system is designed to automate the battery swapping and charging process for drones. It integrates advanced sensor technology with machine learning to ensure efficient and reliable drone operation.

Prerequisites

Hardware:

  • Arduino Board
  • 2x IR Sensors
  • 2x Piezo Electric Sensors
  • 2x 20x4 LCD Display
  • Voltage Divider Circuit
  • INA219 Sensor

Software:

  • Arduino IDE
  • Python Environment (Recommend Anaconda or Jupyter Notebook or Google Colab)
  • Git (for version control)

Installation and Setup

Clone the Repository

First, clone the repository to your local machine using:

gh repo clone mauliksavalia/CS682

Hardware Setup

  1. Connect the 2 IR sensors to the Arduino in the arduino pin 4 & 5.
  2. Connect the Piezo sensors to the same arduino in the arduino pin A0 and A1
  3. Connect the INA219 sensor and 20x4 LCD to the Arduino board for measuring current.
  4. Connect the voltage divider circuit and 20x4 LCD to the Arduino board.
  5. Set up the 20x4 LCD display with the Arduino.

Software Setup

  1. Install Arduino IDE from the official website and upload the respective .ino files to the Arduino board.
  2. Install Anaconda or any Python environment that supports Jupyter Notebooks.

Dependencies for Machine Learning Module

Ensure you have the following Python libraries installed:

  • Pandas: for data manipulation and analysis.
  • NumPy: for numerical operations.
  • Matplotlib: for plotting graphs.
  • Scikit-Learn: for data preprocessing and machine learning.
  • TensorFlow Keras: for neural network models.

You can install these libraries using pip:

pip install pandas numpy matplotlib scikit-learn tensorflow

Running the Project

Running Sensor Modules

IR & Piezo Electric Sensors Integration Setup:

  1. Connect the IR and Piezo Electric sensors to the Arduino as required.
  2. Upload the 2IRand2PiezoSensorsWithLCD.ino file to the Arduino.
  3. Open the Serial Monitor to view the sensor readings.

Current Sensor Setup:

  1. Connect the Current sensor to the Arduino as required.
  2. Upload the Display_Power_T2.ino file to the Arduino.
  3. Observe the sensor readings on the Serial Monitor or LCD display.

Voltage Sensors Setup:

  1. Connect the Voltage sensors to the Arduino as required.
  2. Upload the Display_T1.ino file to the Arduino.
  3. Observe the sensor readings on the Serial Monitor or LCD display.

LCD Display:

Ensure the LCD is connected properly to the Arduino.

Running the Machine Learning Module

  1. Open the Notebook: Navigate to CS682_Battery_Recharging_ML.ipynb and open it in your Jupyter environment, also upload synthetic_data_5.csv file to access the data.
  2. Run the Code: Execute the cells sequentially, ensuring that each library loads successfully and the data paths are correctly set.

The notebook will guide you through the following steps:

  • Data Loading and Preprocessing
  • Data Normalization
  • Model Building and Training
  • Evaluation and Prediction
  • Result Visualization

Visualizing Results

The project includes scripts for visualizing sensor readings and machine learning predictions. Ensure matplotlib is installed and follow the instructions in the notebook to plot graphs.

Troubleshooting

If you encounter any issues:

  • Check that all hardware connections are secure.
  • Ensure all software dependencies are correctly installed.
  • Refer to the Arduino and Python console logs for error messages.
