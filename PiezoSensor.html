Piezo Electric Sensors

This section contains documentation for various Piezo Electric sensors combinations used in our projects. Click on any sensor to learn more about its specifications, setup and usage.

  • Piezo Electric Sensor - Documentation for piezo sensor used for pressure detection.
  • 2 Piezo Electric Sensor - Documentation for 2 piezo sensors used for impact detection where it got activated when both sensors are activated simultaneously.
  • 4 Piezo Electric Sensor - Documentation for 4 piezo sensors used for impact detection where it got activated when all 4 sensors are activated simultaneously.
  • Piezo Electric Sensor with LCD - Documentation for piezo sensor used for pressure detection where LCD is also attached which shows Piezo sensor activation..

Code Documentation

Functions:

  • setup(): Initializes the sensors and LCD display. Sets the initial message on the LCD.
  • loop(): Continuously checks the sensor readings and updates the LCD and Serial Monitor with the drone detection status.

Classes:

  • LiquidCrystal_I2C: Used for communicating with the LCD. Requires the I2C address of the LCD, number of columns, and number of rows.

Variables:

  • piezoSensorPin1, piezoSensorPin2, piezoSensorPin3, piezoSensorPin4: int pins connected to Piezo sensors.
  • piezoThreshold: int value above which piezo is considered "triggered".
  • lcd: LiquidCrystal_I2C object for the LCD display.

Constants:

  • Sensor pins (piezoSensorPin1, piezoSensorPin2, piezoSensorPin3, piezoSensorPin4) are assigned to specific board pins and are not meant to change throughout the program.
  • piezoThreshold is set based on sensitivity requirements and may be adjusted for different environments or sensors.

External Libraries:

  • LiquidCrystal_I2C: Handles the interfacing with the LCD display over the I2C communication protocol.
