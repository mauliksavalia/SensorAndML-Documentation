Piezo Pressure Sensor with LCD Code Documentation

Back to Main Page

Code Overview

This code demonstrates the use of a piezo sensor to detect pressure. If pressure is detected above a certain threshold, it will display a message on an LCD screen and send a message through the Serial monitor. If no pressure is detected, it will display a different message on the LCD screen.

Required Libraries

LiquidCrystal_I2C.h - This library allows communication with the LCD screen using the I2C protocol. If you haven't already, make sure to download and install this library.

Global Variables

  • lcd - An object of the LiquidCrystal_I2C class, used to interact with the LCD.
  • sensorPin - The analog pin connected to the piezo sensor.
  • threshold - The threshold value used to detect pressure.

setup() Function

  1. Initialize the Serial communication with a baud rate of 9600.
  2. Set the sensorPin as an INPUT pin.
  3. Initialize the LCD screen using the init() function.
  4. Enable the backlight of the LCD screen
  5. Print the string "Piezo Sensor:" on the first line of the LCD screen.

loop() Function

  1. Read the analog voltage from the sensorPin and store it in the variable sensorValue.
  2. Check if the value of sensorValue is greater than the threshold.
    • Print the string "Pressure detected!" on the Serial monitor.
    • Enter a while loop, which will run as long as the analog voltage from the sensorPin is above the threshold.
    • Inside the while loop, set the cursor of the LCD screen to the start of the second line.
    • Print the string "Drone Detected!" on the second line of the LCD screen.
  3. Add a small delay of 200 milliseconds.

Additional Notes

This code assumes the LCD screen is connected using the I2C protocol at address 0x27, with 16 columns and 2 rows. Adjust the parameters in the lcd object initialization if your LCD screen has different specifications.
