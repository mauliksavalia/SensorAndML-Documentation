IR Sensor Pins Documentation

This code is used to read the values of four IR sensors and determine if there is an object detected by all four sensors simultaneously. The pins used for each IR sensor are defined as follows:

  • irSensor1 is connected to pin 2
  • irSensor2 is connected to pin 3
  • irSensor3 is connected to pin 4
  • irSensor4 is connected to pin 5

A distance variable is defined as a float and set to a value of 1

Setup

In the setup() function, the pins for each IR sensor are set as input using the pinMode() function. The serial communication is also initialized with a baud rate of 9600 using Serial.begin().

LOOP

In the loop() function, the code reads the values of each sensor using the digitalRead() function. The value is compared to the distance variable and stored in separate bool variables for each sensor:

  • sensor1State stores the state of irSensor1
  • sensor2State stores the state of irSensor2
  • sensor3State stores the state of irSensor3
  • sensor4State stores the state of irSensor4

If all four sensors are triggered simultaneously (i.e., all bool variables are true), the code prints "Object detected by all sensors!" to the serial monitor. Otherwise, it prints "No object detected."

A delay of 300 milliseconds is added using the delay() function before the loop repeats. This delay is necessary to avoid rapid and unnecessary sensor readings.
