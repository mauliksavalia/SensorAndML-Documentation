SETUP

  • Connect the voltage source to pin A0 on the Arduino board.
  • Upload the code to the Arduino board.

PINS

volDetPin - Holds the pin number to which the voltage source is connected (A0 in this case).

CONSTANTS

  • shldredfactor - Holds the factor used to reduce the voltage value calculated.
  • vcc - Holds the reference voltage value

VARIABLES

  • vin - Holds the calculated input voltage value.
  • vout - Holds the voltage value obtained from the analog pin.
  • voltageDetVal - Holds the analog value read from the voltage detection pin

FUNCTIONS

loop()

  • Contains the main code that runs repeatedly.
  • Reads the analog value from volDetPin and assigns it to voltageDetVal.
  • Calculates vout by dividing voltageDetVal by 1024 and multiplying it by vcc
  • Calculates vin by multiplying vout by shldredfactor.
  • Prints the value of vin with a message indicating it is a voltage value.
  • Delays execution for 1000 milliseconds (1 second).
