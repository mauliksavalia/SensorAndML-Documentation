Code Documentation

Pin Definitions

  • irSensorPin1: Pin 4 on the Arduino Mega 2560, used for the first IR sensor.
  • irSensorPin2: Pin 5 on the Arduino Mega 2560, used for the second IR sensor.
  • piezoSensorPin1: Pin A0 on the Arduino Mega 2560, used for the first piezo sensor.
  • piezoSensorPin2: Pin A1 on the Arduino Mega 2560, used for the second piezo sensor.

Sensor Thresholds

piezoThreshold: Threshold value used to determine whether the piezo sensors are triggered or not.

LCD Display Setup

lcd: LiquidCrystal_I2C object with the address 0x27, representing the LCD display.
LCD display has 16 columns and 2 rows.

void setup()

Configures the pin modes for all sensors and initializes the LCD display.
Sets the cursor position on the LCD display to the top-left corner (0, 0).
Prints "System Ready" on the LCD display.
Starts serial communication at a baud rate of 9600.

void loop()

Continuously executes the following steps:
Reads the digital values of both IR sensors and stores the status as boolean values.
Reads the analog values of both piezo sensors.
Determines the status of the piezo sensors based on the threshold value.
Checks if all four sensors are triggered.
If all four sensors are triggered, prints "Drone detected!" on the serial monitor and the LCD display.
If not all four sensors are triggered, prints "No Object Detected" on the serial monitor and "No Drone Detected" on the LCD display.
Waits for 300 milliseconds before repeating the loop.
