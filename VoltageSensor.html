Code Overview

This script is designed to measure voltage using a simple voltage divider connected to an analog input and display the measured voltage on both an LCD and Serial Monitor.

Functions:

  • setup(): Initializes the LCD display and sets up the Serial communication. Displays the initial message on the LCD.
  • loop(): Continuously reads the voltage from the voltage detector pin, calculates the actual voltage, and updates the LCD and Serial Monitor with the current voltage reading every second.

Variables:

  • volDetPin: Analog pin connected to the voltage divider (A0).
  • vin: Actual input voltage being measured.
  • vout: Voltage coming out from the voltage divider.
  • voltageDetVal: Raw analog value read from the voltage detector pin.
  • shldredfactor: Factor to adjust the measured value to the actual voltage (due to voltage divider).
  • vcc: Reference voltage for the Arduino board (usually 5V).
  • lcd: LiquidCrystal_I2C object for interfacing with the LCD.

Constants:

  • volDetPin: Pin number for the voltage detector.
  • shldredfactor: Shield reduction factor, specific to the voltage divider used.
  • vcc: Reference voltage for the Arduino (5V).

External Libraries:

  • LiquidCrystal_I2C.h: Library for controlling the LCD via I2C.
