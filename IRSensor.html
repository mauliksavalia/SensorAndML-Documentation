IR Sensors

This section contains documentation for various IR sensors combinations used in our projects. Click on any sensor to learn more about its specifications, setup and usage.

  • IR Sensor - Documentation for infrared sensor used for proximity detection.
  • IR2Sensor - Documentation for 2 infrared sensors used for proximity detection which gets active when both sensors are activated simultaneously.
  • IR4Sensor - Documentation for 4 infrared sensors used for proximity detection which gets active when all 4 sensors are activated simultaneously.
  • IRSensorWithLCD - Documentation for IR sensor used to measure proximity detection where LCD is also attached which shows IR sensor activation.

Code Documentation

Functions:

  • setup(): Initializes the sensors and LCD display. Sets the initial message on the LCD.
  • loop(): Continuously checks the sensor readings and updates the LCD and Serial Monitor with the drone detection status.

Classes:

  • LiquidCrystal_I2C: Used for communicating with the LCD. Requires the I2C address of the LCD, number of columns, and number of rows.

Variables:

  • irSensorPin1, irSensorPin2, irSensorPin3, irSensorPin4: int pins connected to IR sensors.
  • lcd: LiquidCrystal_I2C object for the LCD display.

Constants:

  • Sensor pins (irSensorPin1, irSensorPin2, irSensorPin3, irSensorPin4) are assigned to specific board pins and are not meant to change throughout the program.

External Libraries:

  • LiquidCrystal_I2C: Handles the interfacing with the LCD display over the I2C communication protocol.
