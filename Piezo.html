Piezo Pressure Sensor Code Documentation

Back to Main Page

This code is used to detect pressure applied to a piezo sensor connected to an Arduino board. If the pressure exceeds a certain threshold, a message will be printed to the serial monitor.

PREREQUISITES

To use this code, you will need the following

  • Arduino board
  • Piezo sensor
  • Arduino IDE

CIRCUIT

Connect the piezo sensor to the Arduino board as follows:

  • Connect the positive (+) pin of the piezo sensor to the A0 pin on the Arduino board.
  • Connect the negative (-) pin of the piezo sensor to the ground (GND) pin on the Arduino board.

CODE

const int sensorPin = A0; // Pin connected to piezo sensor
const int threshold = 10; // Threshold value to detect press
void setup() {
Serial.begin(9600); // Initialize serial
pinMode(sensorPin, INPUT); // Set pin as input
}

void loop() {
int sensorValue = analogRead(sensorPin); // Read analog voltage
// Check if pressure is applied
if (sensorValue > threshold) {
Serial.println("Pressure detected!");
}

delay(10); // Small delay
}

USAGE

  1. Connect the Arduino board to your computer using a USB cable.
  2. Open the Arduino IDE.
  3. Copy the code into a new sketch.
  4. Upload the sketch to the Arduino board.
  5. Open the serial monitor.
  6. Apply pressure to the piezo sensor
  7. If the pressure exceeds the threshold, you should see the message "Pressure detected!" printed to the serial monitor.
