IR and Piezo Electric Sensors Integration with LCD Code Documentation

DESCRIPTION

This code is used to monitor the status of an infrared sensor and a piezo sensor. It uses a LiquidCrystal I2C display to provide visual feedback and outputs relevant messages via the Serial monitor. The code detects the presence of a drone by checking if both sensors are triggered simultaneously.

LCD DISPLAY SETUP

lcd: An instance of the LiquidCrystal_I2C class for controlling the LCD display. It is initialized with the I2C address of the LCD (0x27) and the LCD dimensions (16 columns x 2 rows).

SETUP FUNCTION

The setup function is called once when the microcontroller starts. It performs the following tasks:

  • Sets the pinMode of irSensorPin and piezoSensorPin to INPUT.
  • Initializes the LCD display.
  • Turns on the backlight of the LCD.
  • Begins communication with the Serial monitor at a baud rate of 9600
  • Prints "System Ready" on the first line of the LCD display.

LOOP FUNCTION

  • Reads the status of the infrared sensor and stores it in the irStatus variable.
  • Reads the value of the piezo sensor and stores it in the piezoReading variable.
  • Determines the status of the piezo sensor based on the piezoThreshold. If the reading is above the threshold, piezoStatus is set to true.
  • Checks if both the infrared sensor and the piezo sensor are triggered simultaneously. If they are, it checks if this is the first time they are triggered since the last time they were not triggered. If so, it prints "Motion detected!" on the Serial monitor and "Drone Detected" on the second line of the LCD display.
  • If either the infrared sensor or the piezo sensor is not triggered, it checks if either of them was triggered in the previous loop iteration. If so, it prints "No Object Detected" on the Serial monitor and "No Drone Detected" on the second line of the LCD display.
  • Updates the last status variables (lastIrStatus and lastPiezoStatus) with the current sensor statuses.
  • Adds a delay of 300 milliseconds before starting the next loop iteration.
